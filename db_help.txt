Usage: insforge db [options] [command]

Database operations

Options:
  -h, --help                    display help for command

Commands:
  query [options] <sql>         Execute a SQL query against the database
  tables                        List all database tables
  functions                     List all database functions
  indexes                       List all database indexes
  policies                      List all RLS policies
  triggers                      List all database triggers
  rpc [options] <functionName>  Call a database function via RPC
  export [options]              Export database schema and/or data
  import [options] <file>       Import database from a local SQL file
  help [command]                display help for command
